# ⚠️ 管理员面板路径变更通知

## 🔄 重要变更

**生效日期**：2025-11-05

管理员面板的访问路径已更改：

### ❌ 旧路径（已废弃）
```
/admin-login      → 已废弃
/admin-dashboard  → 已废弃
```

### ✅ 新路径（当前使用）
```
/enen  → 管理员面板（包含登录和后台）
```

---

## 📝 需要做什么

### 1. 更新浏览器书签
- 删除旧书签：`/admin-login` 和 `/admin-dashboard`
- 添加新书签：`/enen`

### 2. 清除浏览器缓存
- 按 `Ctrl+Shift+Delete`（Windows）或 `Cmd+Shift+Delete`（Mac）
- 选择"Cookie 和其他网站数据"
- 点击"清除数据"

### 3. 重新登录
- 访问新路径：`/enen`
- 使用管理员账号登录

---

## ✨ 新的访问方式

### 方式 1：直接访问
```
https://your-domain.com/enen
```

### 方式 2：前台导航（推荐）
1. 以管理员身份登录前台
2. 点击右上角"管理面板"按钮（🛡️ 图标）
3. 自动跳转到 `/enen`

---

## ❓ 常见问题

### Q: 为什么访问 `/admin-login` 没有反应？
A: 该路径已废弃，请使用新路径 `/enen`

### Q: 我的书签还是旧路径怎么办？
A: 请更新书签到新路径 `/enen`

### Q: 新路径有什么优势？
A: 
- 更简洁易记
- 统一登录和后台路径
- 提高安全性（非标准路径）
