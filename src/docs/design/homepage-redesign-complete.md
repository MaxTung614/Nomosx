## 🎨 設計系統

### 配色方案

| 顏色名稱 | 色碼 | 用途 | 應用場景 |
|---------|------|------|----------|
| **Deep Black-Blue** | `#0B0C10` | 主背景 | 頁面整體背景 |
| **Card Background** | `#12141A` | 卡片背景 | 所有卡片組件 |
| **Primary Text** | `#EAEAEA` | 主要文字 | 標題、內容文字 |
| **Secondary Text** | `#9EA3AE` | 次要文字 | 描述文字、標籤 |
| **Accent Gold** | `#FFC107` | 主強調色 | Logo、按鈕、圖標 |
| **Neon Blue** | `#00E0FF` | 次強調色 | 懸停光暈、裝飾 |
| **Error Red** | `#FF5252` | 錯誤/折扣 | 折扣標籤、錯誤提示 |

### 色彩使用原則

```css
/* 背景層次 */
背景 → #0B0C10 (深黑藍)
卡片 → #12141A (稍亮)
懸停 → 漸變光暈效果

/* 文字層次 */
主標題 → #EAEAEA (高亮白)
副標題 → #9EA3AE (中灰)
強調文字 → #FFC107 (金色)

/* 互動元素 */
主按鈕 → #FFC107 背景 + #0B0C10 文字
次按鈕 → #00E0FF 邊框 + 透明背景
懸停效果 → 金色/藍色光暈
```

## ✨ 動畫與互動

### 全局動畫

| 元素 | 動畫效果 | 時長 | 觸發方式 |
|------|---------|------|----------|
| Header | 從上滑入 (`y: -100 → 0`) | - | 頁面加載 |

| 特性卡片 | 依序淡入 | 延遲 0.1s * index | 滾動進入 |

### 懸停效果

| 元素 | 懸停效果 | 過渡時長 |
|------|---------|---------|
| Logo | 縮放 1.05 | - |
| 主按鈕 | 縮放 1.05 + 陰影增強 | 300ms |
| 遊戲卡片 | 上升 8px + 縮放 1.02 | 300ms |
| 遊戲圖片 | 縮放 1.1 + 漸變覆蓋 | 500ms |
| 優惠橫幅 | 縮放 1.02 | 300ms |
| 特性卡片 | 上升 5px | 300ms |

## 🔧 技術實現

### 使用的套件

```json
{
  "motion/react": "動畫庫",
  "lucide-react": "圖標庫",
  "sonner@2.0.3": "Toast 通知",
  "ImageWithFallback": "圖片組件"
}
```

### 關鍵技術

1. **Motion 動畫**:
```tsx
<motion.div
  initial={{ opacity: 0, y: 30 }}
  whileInView={{ opacity: 1, y: 0 }}
  viewport={{ once: true }}
  whileHover={{ y: -8, scale: 1.02 }}
/>
```

2. **內聯樣式** (精確控制顏色):
```tsx
style={{ 
  backgroundColor: '#0B0C10',
  color: '#EAEAEA'
}}
```

3. **響應式圖片**:
```tsx
<ImageWithFallback
  src={game.cover}
  alt={game.name}
  className="w-full h-48 object-cover"
/>
```

4. **條件渲染**:
```tsx
{isAuthenticated ? (
  <Avatar />
) : (
  <Button>登入</Button>
)}
```
